# Personal Portfolio Website

A clean, modern, and responsive personal portfolio website to showcase projects and skills. It features smooth scroll animations, an active navigation indicator, and a persistent light/dark theme toggle.

## Features

-   **Responsive Design**: Looks great on all devices, from mobile phones to desktops.
-   **Dynamic Content**: The copyright year in the footer updates automatically.
-   **Scroll Animations**: Elements subtly fade in as they enter the viewport, powered by the `IntersectionObserver` API.
-   **Active Navigation State**: The navigation link corresponding to the current section on the page is highlighted automatically on scroll.
-   **Theme Toggle**:
    -   Switch between a light and a dark theme.
    -   Your theme preference is saved in `localStorage` for future visits.
    -   Automatically detects and applies the user's preferred color scheme (light or dark) on their first visit.

## Technologies Used

-   HTML5
-   CSS3
-   Vanilla JavaScript
